# Dataprovider-Service

Dataprovider desenvolvido em REST para ser utilizado em testes automatizados. 

### Setup

1. Global Dependencies
    * Install [NODE](https://nodejs.org/en/)

2. Dependencies
	* Install npm packages
	```
	$ npm install
	```    

### Start Service

* NPM:
	```
	$ npm start
	```
